# Learnpack

[Learnpack](https://learnpack.co) is an open source learning platform and CLI that runs exercises, auto-grades code, and powers interactive coding curricula.

**Self-hosted open source** means you run it on your own servers (or your school’s) instead of a SaaS. You get full control of data and customization, no vendor lock-in, and the code is free to use and modify under its license.
